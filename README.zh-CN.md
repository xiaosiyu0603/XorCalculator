# 自述文件

### 简介

　　对输入文件按字节异或。刚好可以将某云音乐的缓存转化为可以正常读取的音频文件。

### 原理

　　将文件中每个字节都按位异或`0xA3`并立刻输出（原因见参考资料）。

### 使用方法

　　将文件（例如`.uc`文件）拖到exe上，即在输入文件的目录下生成对应的输出文件。在某些时候需要手动将扩展名从“.mp3”改为“.flac”。
